PseudoJson

Welcome to PseudoJSON, a tool designed to provide developers with fake REST API endpoints for testing and development purposes. Use these endpoints to simulate server responses without needing a backend setup.

Product API

This is a simple API for managing products in a MongoDB database.

GET /products

Returns a list of all products.

GET /products/{id}

Returns a single product by its ID.

POST /products

Adds a new product. Requires product data in JSON format.

PUT /products/{id}

Updates an existing product by its ID. Requires updated data in JSON format.

DELETE /products/{id}

Deletes a product by its ID.
